# Human Detector System

This project implements:

1. A basic Turing Test evaluator
2. A CAPTCHA verification system

## How to Run

pip install -r requirements.txt
python app.py

## Routes

GET /captcha
POST /verify_captcha
POST /turing

## Features
- Basic CAPTCHA generation and verification
- Mini Turing Test evaluator
- Flask-based API system

## Technologies Used
- Python
- Flask
- Git & GitHub
